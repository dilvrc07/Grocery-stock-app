# Grocery Stock Management App: It is a simple Management system built using python and streamlit.
this application helps manage grocery items,monitor stock, and check expiry dates easily.
## Features:
>Login System
>Add Grocery Items
>View Stock details
>Search by barcode
>Update and delete items
>Expiry date alert
>Lottie animation UI
>SMS alert using Twilio
## Technolgies Used:
>python
>streamlit
>pandas
>twilio API'
>Lottie JSON animations
## How to Run:
>Install vscode
>Install required packages
>run the program using:
python -m streamlit run app.py
